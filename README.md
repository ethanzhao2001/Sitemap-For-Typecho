# Sitemap-For-Typecho

用着好的话麻烦给个star，十分感谢！！！

## 功能

1、为`Typecho`生成`sitemap`包含**首页、独立页面、分类、标签、文章**

2、推送百度搜索资源平台，目前仅支持普通收录推送，卑微的我没快速收录权限

3、支持永久链接中的全部参数，包括以下。

```
可用参数: 
{cid} 日志 ID
{slug} 日志缩略名
{category} 分类
{directory} 多级分类
{year} 年
{month} 月
{day} 日
```

~~不支持发布文章自动更新`sitemap`及自动推送，可能会影响文章发布速度所以没有做支持~~

### API说明

| 参数 | 值 | 说明 | 
| ------- | --------- | ----------------- | 
| sitemap | update | 更新sitemap |
| push | main | 推送核心文章 |
| push | all | 推送全部文章 |
| push | new | 推送最新文章 |
| token | API token | 插件中的API token |
